HƯỚNG DẪN SỬ DỤNG
-----------------
1. Mở file bằng https://excalidraw.com
2. Chỉnh sửa nếu cần
3. Export > PNG hoặc SVG
4. Chèn vào Word với kích thước phù hợp (chiều rộng ~12-14cm cho A4)
